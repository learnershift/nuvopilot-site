Campaign Leak Check

Delete Data

Campaign Leak Check's current submitted v1 target stores user-entered campaign checks locally on the user's device. It does not create an account, upload records to a server, or maintain a developer-side user database.

Delete Local Data

  1. Open Campaign Leak Check.
  2. Go to Settings.
  3. Select the clear local data action.
  4. Confirm that saved checks and local intent events are removed.

Export Before Deleting

If you want to keep a copy, use the in-app export action before clearing local data. Export is user-triggered only.

Remote Deletion Requests

Because the current submitted v1 target has no developer-side account or server database, NuvoPilot cannot remotely delete local device data. For privacy questions, contact support@nuvopilot.com and do not include sensitive legal, payment, advertising account, or customer data.

Support Email Records

If a user contacts support by email, the support message is handled through the user's email provider and NuvoPilot's support mailbox. Users can request deletion of support email correspondence by contacting support@nuvopilot.com, subject to legal, security, and abuse-prevention retention needs.
